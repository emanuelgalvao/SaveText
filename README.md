<div align="center">
  <h2>Descrição do Projeto</h2>
  
  <p>Este projeto foi desenvolvido inspirado no site <a href="http://dontpad.com/" target="_blank">dontpad.com</a> que é um site onde é possível guardar textos com base na URL digitada no navegador sem a necessidade de manter algum arquivo na maquina do usuário!</p>
  <p>O objetivo principal do projeto foi a implementação de um sistema de rotas que identifique a URL que foi digitada e busque o texto relacionado em um banco de dados.</p>
  <p>Neste projeto também foi implementada a função de autosalvamento das informações no banco de dados a cada intervalo de tempo!</p>
